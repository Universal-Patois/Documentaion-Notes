# Breakpoints

## Common breakpoints

The following breakpoints are available for use in your application. They are based on the [Bootstrap breakpoints](http://getbootstrap.com/css/#grid-options).

| Breakpoint | Description | Min width | Max width |
| ---------- | ----------- | --------- | --------- |
| xs | Extra small devices (phones) | 0px | 576px |
| sm | Small devices (tablets) | 576px | 768px |
| md | Medium devices (desktops) | 768px | 992px |
| lg | Large devices (desktops) | 992px | 1200px |
| xl | Extra large devices (desktops) | 1200px | Infinity |
